 Text-to-Maths-Problem-Solver
 Project Description:

Text-to-Maths-Problem-Solver is an AI-powered application that takes math problems written in plain English (text form) and automatically converts them into solved mathematical expressions with step-by-step solutions. 

Using LangChain, LLM models (like Groq + Llama3), and Streamlit, this tool can understand natural language, interpret mathematical intent, and produce accurate results — all in real time! 

 Key Features:

🔹  Natural Language Understanding:
Converts complex word problems into solvable math expressions.

🔹  Step-by-Step Solutions:
Displays each calculation step clearly for better understanding.

🔹  Interactive UI with Streamlit:
Simple and clean interface for entering and viewing results instantly.

🔹  LLM-Powered Reasoning:
Utilizes LLMs to interpret tricky phrases like “twice the sum of…” or “half of the difference between…”.

🔹  Supports Multiple Topics:
Arithmetic  | Algebra  | Geometry  | Trigonometry  | Word Problems 

 Tech Stack:

 LangChain – for chaining and prompting

 ChatGroq / Llama3 – for powerful AI reasoning

 Streamlit – for interactive front-end

 Python – core logic and data handling

 Use Case Example:

Input: “What is twice the sum of 8 and 5?”
Output:

2 × (8 + 5) = 2 × 13 = 26
Answer: 26 

 Why It Stands Out:

 Converts natural text directly into mathematical reasoning
 Great for students, educators, and developers learning AI + math integration
 Instant computation with LLM + LangChain workflow
